# CLI Reference

## Quick Reference

| Option | Description |
|--------|-------------|
| `--version`, `-v` | Print version and exit |
| `--help`, `-h` | Print help and exit |
| `--config=<path>` | Use alternate config file |
| `--json` | Output JSON format (where supported) |

For the full command reference, see [Commands](../cli/commands.md).
